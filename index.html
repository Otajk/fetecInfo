O que é o Scratch ?

O Scratch é uma plataforma de programação visual que permite a criação de projetos interativos, como jogos, animações e histórias. É amplamente utilizada no ensino de programação para jovens, pois seu ambiente é intuitivo e acessível. Os usuários arrastam e soltam blocos de código para construir suas aplicações, o que facilita a compreensão de conceitos básicos de programação.
Na ETEC Darcy Pereira de Moraes, os alunos utilizaram o Scratch para desenvolver jogos como parte da FETEC. Essa feira é uma oportunidade para os estudantes apresentarem suas criações e aplicarem o que aprenderam em sala de aula. Os jogos geralmente refletem a criatividade e a colaboração dos alunos, e podem variar desde plataformas simples até jogos de aventura mais complexos.
